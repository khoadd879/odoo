# SOP-SHIP-007 — Dangerous Goods Declaration Procedure

**Document ID:** SOP-SHIP-007
**Version:** 2.5
**Effective Date:** 2026-02-01
**Owner:** Shipping Compliance Lead — John Kaupa
**Visibility:** STAFF ONLY

## 1. Purpose

Ensure that any cargo classified as Dangerous Goods under the IMO International Maritime Dangerous Goods (IMDG) Code is properly declared, packaged, marked, and documented before acceptance on a Consort vessel. Failure to follow this procedure is a criminal offence under PNG Maritime Safety Act and may invalidate vessel insurance.

## 2. When this SOP applies

This procedure applies when ANY of the following are true:

- The cargo description includes any UN number (e.g. "UN1230").
- The Safety Data Sheet (SDS) shows any GHS hazard pictogram.
- The cargo is one of the 9 IMO classes: explosives, gases, flammable liquids, flammable solids, oxidizers, toxic, radioactive, corrosives, miscellaneous.
- The client answers YES to any question on the DG screening form.

When in doubt, treat the cargo as dangerous goods and call the Compliance Lead before accepting the booking.

## 3. Required documents for DG cargo

In addition to the standard FCL/LCL documents (see SOP-SHIP-004), the client MUST provide:

1. **Dangerous Goods Declaration (DGD)** — IMO standard format, signed by a competent person. Must include UN number, proper shipping name, hazard class, packing group, flash point (if applicable), and marine pollutant status.
2. **Safety Data Sheet (SDS)** — current within 5 years, 16-section format.
3. **Container Packing Certificate** — confirming the dangerous goods have been properly stowed and secured inside the container.
4. **Emergency contact** — 24/7 phone number of a person knowledgeable about the cargo.

## 4. Booking workflow for DG cargo

| Step | Action | Owner |
|------|--------|-------|
| 1 | Client submits DGD with booking request | Client |
| 2 | Compliance Lead reviews DGD and SDS for completeness | Compliance |
| 3 | Operations verifies vessel is approved for the IMO class | Operations |
| 4 | Special stowage position booked on vessel plan | Operations |
| 5 | Master signs acceptance; DGD filed with vessel manifest | Master + Operations |
| 6 | Notify party briefed on emergency response | Operations |

## 5. Common DG cargoes in PNG

- **UN1268** — Petroleum distillates (diesel, fuel oil) — Class 3
- **UN1950** — Aerosols — Class 2.1
- **UN1823** — Sodium hydroxide solid — Class 8
- **UN1789** — Hydrochloric acid — Class 8
- **UN1006** — Argon compressed — Class 2.2
